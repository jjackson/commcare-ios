## Key Technical Decisions

1. **XFormParser port over wrapper**: Rather than creating a wrapper, ported the full 2853-line parser to commonMain with a cross-platform DOM abstraction. This gives iOS native form parsing.

2. **Golden file pattern**: JVM oracle generates expected output once → checked into repo → both platforms compare in commonTest. Avoids running kxml2-dependent oracle on iOS.

3. **Namespace enumeration on PlatformXmlParser**: KXmlParser doesn't expose xmlns attributes through `getAttributeCount()` when namespace processing is enabled. Added `getNamespaceCount/Prefix/Uri` methods to preserve namespace declarations through the DOM.

4. **test_constraints.xml rename**: Upstream's `src/test/resources/test_constraints.xml` collided with our `src/commonTest/resources/test_constraints.xml` on the JVM classpath. Renamed to `test_field_list_constraints.xml`.
